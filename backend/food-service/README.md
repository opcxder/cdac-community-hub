# Food Service

This is the Food Service for the CDAC project, built with Spring Boot.

## Prerequisites
- Java 17
- MySQL running on localhost:3306 with database `food_db`
- Maven

## Setup
1. Clone the repository.
2. Update `src/main/resources/application.properties` with your MySQL credentials (default: root/password).
3. Run `mvn clean install`.
4. Run `mvn spring-boot:run` or just run `FoodServiceApplication.java`.
